# 基于SSM的个人博客管理系统 - 项目总结

## 项目概述

本项目是一个完整的基于SSM（Spring + Spring MVC + MyBatis）框架开发的个人博客管理系统，实现了博客文章的发布管理、用户评论互动、留言板等核心功能。系统采用前后端分离的设计思路，提供了用户友好的前台展示界面和功能完善的后台管理系统。

## 技术架构

### 后端技术栈
- **框架**: Spring 5.3.21 + Spring MVC + MyBatis 3.5.10
- **数据库**: MySQL 8.0.29
- **连接池**: Druid 1.2.11
- **日志**: Logback 1.2.11
- **构建工具**: Maven 3.8+
- **JDK版本**: JDK 8+

### 前端技术栈
- **视图技术**: JSP + JSTL
- **UI框架**: Bootstrap 5.1.3
- **JavaScript库**: jQuery 3.6.0
- **图标**: Font Awesome 6.0.0

## 文件结构

```
src/
├── main/
│   ├── java/com/blog/
│   │   ├── controller/     # 控制层 (2个文件)
│   │   ├── service/        # 业务层 (4个接口 + 4个实现)
│   │   ├── dao/           # 数据访问层 (5个接口)
│   │   ├── entity/        # 实体类 (5个实体)
│   │   ├── util/          # 工具类 (2个工具类)
│   │   └── interceptor/   # 拦截器 (1个拦截器)
│   ├── resources/
│   │   ├── config/        # 配置文件 (4个配置)
│   │   ├── mapper/        # MyBatis映射 (5个映射文件)
│   │   ├── blog_system.sql # 数据库脚本
│   │   └── logback.xml    # 日志配置
│   └── webapp/
│       ├── WEB-INF/
│       │   ├── views/     # JSP视图 (5个页面)
│       │   └── web.xml    # Web配置
│       └── css/           # 样式文件
├── README.md              # 项目说明
├── DEPLOYMENT.md          # 部署文档
├── demo-data.sql          # 演示数据
├── start.sh              # 启动脚本
└── pom.xml               # Maven配置
```
